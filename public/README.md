# Static Marketing Site

This directory contains the static marketing pages for Parlay Streak, including:
- Landing page (`index.html`)
- Legal pages (`legal/`)
- Company pages (`company/`)

## Deployment

This directory can be deployed as a static site on Render or any static hosting service.

### Render Deployment

1. Create a new Static Site service on Render
2. Set the root directory to `public`
3. Build command: (leave empty or use `echo "No build needed"`
4. Publish directory: `public`

The `_redirects` file ensures proper routing for all pages.
